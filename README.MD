# Sprint 1 — Task 03: Arrays

## 📄 Description

This repository contains the exercises for **Sprint 1 — Task 03**.

In this task, different operations with **arrays in PHP** were practiced, including indexed arrays, associative arrays, and the use of built-in functions.

---

## 🎯 Objectives

* Create and use arrays
* Perform operations with arrays
* Work with associative arrays
* Use PHP built-in functions for array manipulation

---

## 🛠 Technologies

* PHP

---

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/M3lgone/task-s1-03.git
```

---

## ▶️ How to Run

You can run each level independently using PHP's built-in server.

### Example:

```bash
php -S localhost:8000 -t level-1
```

Then open in your browser:

```
http://localhost:8000
```

👉 Replace `level-1` with `level-2` or `level-3` depending on what you want to test.
