# Sprint 1 — Task 03: Arrays

## 📄 Description

This repository contains the exercises for **Sprint 1 — Task 03**.

In this task, different operations with **arrays in PHP** were practiced, including indexed arrays, associative arrays, and the use of built-in functions.

---

## 🎯 Objectives

* Create and use arrays
* Perform operations with arrays
* Work with associative arrays
* Use PHP built-in functions for array manipulation
